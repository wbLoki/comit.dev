## Usage

### Generate commit message

To generate a commit message, stage your changes and run the following command:

<CodeSnippet code='comit' hasCopy />

you will be prompted to select a commit message

<CodeSnippet
    code={`Use the arrow keys to navigate: ↓ ↑ → ←
? Select commit message:
  > Refresh
    feat: add pagination support to /users endpoint
    fix: resolve 500 error when POST /login is called without payload
    chore: refactor API route handlers for better modularity
    docs: update API reference with new pagination and auth examples`}
/>

### Chat live with you favorite AI assistant **_for now only openAI is supported_**

To chat live with you favorite AI assistant, run the following command:

<CodeSnippet code='comit --live' hasCopy />

you will be prompted to chat with your favorite AI assistant and maintain a conversation
